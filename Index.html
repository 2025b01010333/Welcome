In HTML itself, you don’t have a direct concept of an array like in programming languages such as JavaScript or Java. HTML is a markup language, so it’s used to structure content. However, you can represent array‑like data using lists (ordered or unordered) or by combining HTML with JavaScript, where arrays are actually defined and manipulated.
